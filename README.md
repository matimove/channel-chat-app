# channel-chat-app

# Program features:

# The user can create an account and log in to the application.

# The user can post short text posts. In addition, the user can edit and delete the posts they have added.

# The user can view the posts added to the application. The user can see both the posts they have added written themselves and those posted by other users.

# The user can search for posts using a keyword. The user can search for both the posts they have added themselves and those posted by other users.

# The application has user pages that display statistics for each user and the posts made by that user.

# The user can assign one or more channels for the post besides the main channel (i.e @sport @dogs @food). The post channels are stored in the database.

# Users can up vote and down vote posts, comment on posts and comments can be up and downvoted. Posts and comments can also have emoji reactions on them.
